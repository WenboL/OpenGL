Button 'w' change the terrain between line mode and polygon mode.
Button 'r' reset the terrain.
Button 'h' can grow your current terrain mountains.
Button 'q' Quit, it kills the quite terrain.

Button x,s,w,z to turn around,
	eg. if u want to turn right, press x-> s-> a->z. (press each one till the camera stop moving then next button)	
	eg. if u want to turn left, press s-> x-> a-> z.(press each one till the camera stop moving then next button)	



Button "," "." up, down, left, right  can make you move on the terrain. (changes x,y,z coordinates)
	, : 		z value decrease
	. : 		z value increase
	LEFT:	x value decrease
	RIGHT:	x value increase
	UP :		y value decrease
	DOWN:	y value increase
